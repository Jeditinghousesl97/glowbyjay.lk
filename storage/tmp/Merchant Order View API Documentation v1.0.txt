Payment Method Integration  
Merchant Order View API 
Documentation  
V1.0 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
API Endpoint  
 
POST  https://prodapi.paykoko.com/api/merchants/orderView  
 
Description  
This API allows merchants to view payment status by providing the Merchant ID, Plugin Name, 
and Order ID. The request must be signed with the merchant's private key for security.  
The payment status is retrieved using the orderId  specified in the OrderCreat e request. To 
ensure reliable tracking and retrieval of order details, the orderId  from the eCommerce 
platform must be handled uniquely across the system. Each new KOKO payment request 
should come with a new, unique orderId  to prevent conflicts and ensure that each transaction 
is properly tracked and managed.
